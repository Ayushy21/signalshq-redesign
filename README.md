# SignalsHQ Redesign

A premium, state-of-the-art landing page redesign for SignalsHQ, an AI platform built for CPA firms.

## Overview

This project is a React-based landing page built with Vite, TypeScript, and Tailwind CSS. It features a complete overhaul from a minimalist light theme to a highly textured, premium dark theme designed to evoke the feeling of top-tier developer tools and enterprise software.

### Key Features
- **Dark Theme:** Deep slate backgrounds (`slate-950`) with glowing mesh gradients (teal and indigo) for ambient light.
- **High-Fidelity Cards:** Frosted glass aesthetics (`backdrop-blur-md`) with deep shadows and subtle borders to give UI elements physical presence.
- **Dynamic Animations:** Scroll-triggered framer-motion animations that give the page life without being overwhelming.
- **Responsive Design:** Fully responsive layout with custom mobile navigation and optimized horizontal-to-vertical pipelines.

## Tech Stack
- React 18
- TypeScript
- Vite
- Tailwind CSS
- Framer Motion
- Lucide React (Icons)

## Getting Started

First, install the dependencies:
```bash
npm install
```

Then, run the development server:
```bash
npm run dev
```
